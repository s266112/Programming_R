Per la cluster analysis (customer personality analysis), le variabili più adatte sono quelle che descrivono profilo e comportamento del cliente:

  • Income (disponibilità economica)
  • Age (da calcolare da Year_Birth)
  • Total_spending (spesa totale)
  • Total_purchases (numero totale di acquisti)
  • NumWebVisitsMonth (frequenza di visita)

🔹 Creazione nuove variabili

marketing_clean$Total_spending <- marketing_clean$MntWines + 
                                  marketing_clean$MntFruits +

                                  marketing_clean$MntFishProducts +

🔹 Dendrogrammi

  • Per ciascun metodo rappresentare il dendrogramma; cosa si può osservare?

# Complete linkage

# Ward
plot(hc_ward, labels = FALSE, main = "Ward", xlab = "", sub = "")

💥 Risposta:

  • Single linkage: produce cluster “a catena” (chaining effect), con osservazioni collegate progressivamente. I cluster risultano poco compatti e spesso sbilanciati.

  • Complete linkage: tende a formare cluster più compatti e bilanciati rispetto al single linkage. Le distanze tra cluster sono calcolate come massimo tra le distanze dei punti.

  • Average linkage: rappresenta un compromesso tra single e complete. I cluster sono generalmente ben bilanciati e compatti.

  • Ward: minimizza la varianza all’interno dei cluster. Produce cluster sferici e compatti, ed è spesso il metodo più interpretabile.

set.seed(123)

wss <- sapply(1:10, function(k) {
  kmeans(cluster_scaled, centers = k, nstart = 25)$tot.withinss
})

plot(1:10, wss,
     type = "b",
     pch = 19,
     col = "steelblue",
     xlab = "Numero di cluster K",
     ylab = "Within-cluster sum of squares",
     main = "Metodo del gomito")

# Silhuoette score
library(cluster)
sil_values <- numeric(9)
for (k in 2:10) {
  km_temp <- kmeans(cluster_scaled, centers = k, nstart = 25)
  sil <- silhouette(km_temp$cluster, dist(cluster_scaled))
  sil_values[k - 1] <- mean(sil[, 3])
}

plot(2:10, sil_values,
     type = "b",
     pch = 19,
     col = "steelblue",
     xlab = "Numero di cluster K",
     ylab = "Silhouette media",
     main = "Silhouette score")

💥 Risposta:

  • Metodo del gomito: il “gomito” si individua a K = 2 (riduzione marcata della WCSS fino a 2, poi la curva si appiattisce).
  • Silhouette score: il valore massimo si raggiunge per K = 2 (≈ 0.37), indicando la migliore separazione tra i gruppi.
