2. Ricodifica delle variabili

  1. Dopo aver verificato la presenza di NA (come sono rappresentati e quanti sono per variabile?), rimuovere dal dataset le osservazioni contenenti almeno un valore mancante;

  2. Alcune variabili qualitative possono essere trasformate in factor direttamente, altre no; perché?

  3. Dopo aver trasformato le variabili qualitative in factor, riordinare i livelli ove necessario;

  4. Si consideri la variabile study_hours_per_day e si aggiunga al dataset una nuova variabile study_time che abbia, per ogni osservazione:

  • low se study_hours_per_day è minore di 3,
  • medium se è compresa tra 3 e 6,
  • high altrimenti.
  1. Si consideri la variabile attendance_percentage e si aggiunga al dataset un nuova variabile attendance che abbia, per ogni osservazione:
  • low se attendance_percentage è minore di 60,
  • medium se è compresa tra 60 e 80,
  • high altrimenti.

# Rimuovo righe con NA
studenti_clean <- na.omit(studenti)

# Trasformo variabili in factor
studenti_clean$gender <- as.factor(studenti_clean$gender)
studenti_clean$internet_access <- as.factor(studenti_clean$internet_access)
studenti_clean$extra_classes <- as.factor(studenti_clean$extra_classes)
studenti_clean$parent_education <- as.factor(studenti_clean$parent_education)

# Trasformo grade in factor ORDINATO
studenti_clean$grade <- factor(
  studenti_clean$grade,
  levels = c("F","D","C","B","A"),
  ordered = TRUE
)

# Nuova variabile study_time
studenti_clean$study_time <- cut(
  studenti_clean$study_hours_per_day,
  breaks = c(-Inf, 3, 6, Inf),
  labels = c("low", "medium", "high")
)

# Nuova variabile attendance
studenti_clean$attendance <- cut(
  studenti_clean$attendance_percentage,
  breaks = c(-Inf, 60, 80, Inf),
  labels = c("low", "medium", "high")
)

💥 Risposte

a): I valori mancanti sono rappresentati come NA. Sono presenti in:
- study_hours_per_day (87),
- attendance_percentage (106),
- sleep_hours (104)
- overall_score (104).

Le osservazioni contenenti almeno un valore mancante sono state rimosse dal dataset.

b): Le variabili qualitative possono essere trasformate in factor quando rappresentano categorie. Tuttavia, alcune variabili (come grade) non possono essere trasformate direttamente perché sono ordinali e richiedono un ordine tra i livelli.

c): Dopo la trasformazione in factor, i livelli devono essere riordinati per le variabili ordinali (ad esempio grade), in modo da rispettare l’ordine logico (es. A > B > C > D).

d): È stata creata la variabile study_time che classifica study_hours_per_day in low, medium e high.

e): È stata creata la variabile attendance che classifica attendance_percentage in low, medium e high.

🔹 a) Indici di posizione, dispersione e simmetria

# Funzione per calcolare indici principali
indici_variabile <- function(x) {
  c(
    media = mean(x),
    mediana = median(x),
    minimo = min(x),
    massimo = max(x),
    primo_quartile = quantile(x, 0.25),
    terzo_quartile = quantile(x, 0.75),
    varianza = var(x),
    deviazione_standard = sd(x),
    IQR = IQR(x),
    simmetria = mean((x - mean(x))^3) / sd(x)^3
  )
}

🔹 h) Distribuzione overall_score

# Istogramma + densità
hist(studenti_clean$overall_score,
     probability = TRUE,
     col = "lightblue",
     main = "Distribuzione overall_score")

lines(density(studenti_clean$overall_score), col = "red", lwd = 2)

# linea media
abline(v = mean(studenti_clean$overall_score), col = "red", lwd = 2)

# ECDF
plot(ecdf(studenti_clean$overall_score),
     main = "ECDF overall_score")

# QQ plot
qqnorm(studenti_clean$overall_score)
qqline(studenti_clean$overall_score, col = "red")

💥 Risposte

  • L’istogramma mostra una distribuzione circa simmetrica e unimodale, con forma simile a una distribuzione normale.

  • La curva di densità conferma una buona approssimazione a una distribuzione normale, con valori concentrati intorno alla media.

  • La funzione di ripartizione empirica (ECDF) presenta un andamento regolare e crescente, coerente con una distribuzione continua.

  • Il grafico quantile-quantile (QQ plot) mostra che i punti sono generalmente allineati alla retta, con leggere deviazioni nelle code.

👉 Si può quindi concludere che la distribuzione di overall_score è approssimativamente normale.
